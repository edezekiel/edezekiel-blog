Groupie - My First CLI

November 16, 2018

This week I wrote my first Command Line Interface! It's callled "Groupie," and it helps people get tickets to see their favorite concerts.

Here is the Github link to the source code in case you are interested in taking a look.

Organizing the Run File Using Screens

As you may have noticed during the walkthrough, Groupie has serval "menus" or "screens." Different screens can be accessed at different times throught the execution of the file. Each screen lets the user interact with some aspect of the CLI. For example, the "greeting screen" welcomes the user and provides basic instructions.

These are the different screens:

  • Greeting;
  • Concert list;
  • Individual concert information;
  • Ticketing;
  • Band list;
  • Individual band information; and
  • Exit - allowing the user to exit at any time.

The trickiest part was getting the exit screen and the ticketing screens to work. I wanted to user to be able to exit the prgogram at any time and the buy tickets at several different points.

Formatting - Pastel and Separators

I stylized Groupie using the "pastel" gem. I also created a method called "separators" that was responsible for inserting line breaks between each screen.

ActiveRecord

I used the ActiveRecord gem to streamline several aspects of the program. This gem helps with creating and updating databases, creating associations between classes, and inheriting all kinds of built-in methods.
